# ChatGPT-Discord-Bot
# How to get session?
1. Go to https://chat.openai.com/chat
2. Open Developer Tools and go to Network tab
3. Find the Cookie and the key named "__Secure-next-auth.session-token"
4. Copy the Value paste on session varible.
